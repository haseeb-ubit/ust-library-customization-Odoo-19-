# UST Library Customization & Management System (Odoo 19)

A comprehensive, enterprise-scale management ecosystem built on the **Odoo 19** framework. This system digitizes campus library logistics, automates tracking parameters for resource distribution, handles facilities management, and delivers real-time analytical data visibility via interactive operational dashboards.

## 🚀 Key Modules & Implemented Features

### 📊 Advanced Data Dashboard & Analytics
- **Dynamic Graphical Reporting:** Engineered a data-driven library dashboard utilizing analytical graphs to trace operational metrics, mapping active trends for **borrowed, damaged, and issued books**.
- **Key Performance Indicators (KPIs):** Integrated dynamic data summary cards highlighting strategic inventory intelligence, such as pinpointing **most reviewed** and **least reviewed** educational assets.

### 🏫 Extended Library Facilities Management
- **Resource Logistics:** Developed a tailored, lightweight library management architecture to systematically regulate book registration, checkout validations, and automated returns.
- **Student Study Space Allocation:** Built out an integrated facilities management sub-module to handle booking, availability parameters, and scheduling optimization for **student study rooms**.

---

## 🛠️ Architecture & Core Contributions
The primary engineering effort within the `ust_library` architecture was executed directly across backend structures and presentation layers:

- **Data Layer (`models/`):** Designed and extended database models using Python to establish relational parameters, data validation rules, custom compute functions, and complex schema logic.
- **Presentation Layer (`views/`):** Architected clean XML user interfaces, tailored kanban/tree/form layouts, and interactive dashboard views to prioritize high data legibility and seamless UI/UX workflows.

## 🧰 Technical Stack
- **Framework:** Odoo 19 Framework
- **Languages:** Python (Backend Architecture), XML (Interface Engineering)
- **Database:** PostgreSQL Relational Database Management
- **Design Focus:** Analytical Dashboards & Workflow Optimization
